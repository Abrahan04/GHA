Implementación de CI/CD con GitHub Actions

Este sitio web fue desarrollado como parte del Taller 1 de CI/CD, utilizando GitHub Actions para automatizar la integración y el despliegue continuo del proyecto.

Descripción del Proyecto

El proyecto consiste en la creación de un sitio web estático que se despliega automáticamente en GitHub Pages cada vez que se realiza un cambio en el repositorio principal.

Caso de Uso

Automatizar el proceso de publicación de un sitio web, permitiendo que cualquier actualización realizada en el código se refleje de forma inmediata en el sitio web sin intervención manual.

Tecnologías Utilizadas

  • HTML5
  • Git y GitHub
  • GitHub Actions
  • GitHub Pages
